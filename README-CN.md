## 概述

ETHEREUM-PHP 支持以太坊的 ETH 和 ERC20 数字资产常用的生成地址，发起转账，查询余额，离线签名等功能。

## 快速开始

### 安装

``` php
composer require fenguoz/ethereum-php
```
